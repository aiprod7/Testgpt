# Mini Tetris на Python

Простая мини-игра «Тетрис» на Python без внешних зависимостей. Для графики используется стандартный модуль `tkinter`.

В этой версии добавлена новая цветовая тема **neon sunset**: темный фиолетовый фон, яркие неоновые фигуры, мягкая тень падения и обновленная боковая панель.

## Запуск

```bash
python tetris.py
```

Если в Linux модуль `tkinter` не установлен, поставьте пакет для вашей версии Python, например:

```bash
sudo apt install python3-tk
```

## Управление

- `←` / `→` — двигать фигуру
- `↓` — ускорить падение
- `↑` или `X` — повернуть фигуру
- `Space` — мгновенно сбросить фигуру вниз
- `P` — пауза
- `R` — начать заново

## Что есть в игре

- поле 10×20;
- все классические фигуры;
- поворот фигур;
- тень места падения;
- подсчет очков, линий и уровня;
- ускорение игры с ростом уровня;
- пауза и перезапуск;
- новая яркая цветовая палитра.
